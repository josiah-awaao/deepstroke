TypingDNA specializes in typing biometrics(also known as keystroke dynamics) authentication. 
Our Authentication API can be used by any company that wants to integrate authentication services within their apps. 

TypingDNA REST API can be used with any app. Once you have your userkey and pass you can call the API for the 
following services: match, save, verify, deleteuser, checkuser.

More info & documentation: https://api.typingdna.com/
